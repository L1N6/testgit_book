# Expression

This condition is used to compare conditions with properties in a record row, a form.

\[  ,   ,   ] means “OR”&#x20;

{  ,    , } means “AND”

Criteria modifiers: The following modifiers are available to use when building queries.

* `'='`
* `'<'`
* `'<='`
* `'>'`
* `'>='`
* `'!='`
* `nin`
* `in`
* `contains`
* `startsWith`
* `endsWith`
